# QuizMaster 🧠

A modern and interactive **Quiz Learning Platform** built with React. Test your knowledge across multiple categories with a real-time timer, instant scoring, and a global leaderboard.

![QuizMaster](https://via.placeholder.com/800x400/4f46e5/ffffff?text=QuizMaster)

## ✨ Features

- **Multiple Categories**: General Knowledge, Science, Programming & more
- **Timed Questions**: 20 seconds per question with live timer
- **Instant Feedback**: See correct/incorrect answers immediately
- **Score Calculation**: 10 points per correct answer
- **Progress Tracking**: Question progress bar
- **Leaderboard**: Global leaderboard (saved in localStorage)
- **Responsive Design**: Works perfectly on mobile, tablet & desktop
- **Smooth UI**: Clean dark theme with modern cards and buttons
- **Restart & Play Again**: Easy to retake quizzes

## 🛠️ Tech Stack

- **Frontend**: React 18 + Vite
- **Routing**: React Router v6
- **State Management**: useReducer + Context API
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Ready for Framer Motion (can be added)
- **Data Persistence**: localStorage

## 🚀 Quick Start

### 1. Clone the Project
```bash
git clone git@github.com:mark-j-yadav/Quiz-PlatFrom.git
cd quizmaster
```
### 2. Install Dependencies
```bash
npm install
```
### 3. Run the Development Server
``` bash
run run dev

```
Open http://localhost:5173 to view it in your browser.


### 🎮 How to Use

Home Page: Choose any quiz category (GK, Science, Programming)
Start Quiz: Click on any category card
Answer Questions: Select the correct option within 20 seconds
View Results: See your final score, accuracy, and time taken
Leaderboard: Check top scores from all attempts
